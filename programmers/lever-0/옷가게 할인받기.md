# 옷가게 할인받기

## 문제 설명
머쓱이네 옷가게는 10만 원 이상 사면 5%, 30만 원 이상 사면 10%, 50만 원 이상 사면 20%를 

할인해줍니다. 구매한 옷의 가격 price가 주어질 때, 

지불해야 할 금액을 return 하도록 solution 함수를 완성해보세요.

## 제한사항
- 10 ≤ price ≤ 1,000,000
- price는 10원 단위로(1의 자리가 0) 주어집니다.
- 소수점 이하를 버린 정수를 return합니다.

## 문제풀이
```python
def solution(price):
    answer = 0
    if price >= 500000:
        answer = price * 0.8
    elif price >= 300000:
        answer = price * 0.9
    elif price >= 100000:
        answer = price * 0.95
        # 할인 조건이 없을떼 
    else:
        answer = price    
    # 솟수점은 버려야 하므로 int() 사용
    return int(answer)
```
